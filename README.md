# Expense Tracker

A simple Python console-based Expense Tracker developed using Python. This project allows users to add expenses, view expense records, and calculate total spending through a menu-driven interface.

## Features

✅ Add New Expenses
✅ Store Date, Category, Description, and Amount
✅ View All Expense Records
✅ Calculate Total Spending
✅ Menu-Driven User Interface

## Technologies Used
- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements
